手動インストール手順

基本的なインストール

  1. AviUtl のお部屋から，aviutl???.zip をダウンロードし，適当な解凍ソフトで解凍します．
  2. ダウンロードから vsd_filter_gps_yofukashi???.exe をダウンロードし，W クリックで解凍してください．そしてインストーラが起動しますが，インストーラは終了します．そして vsd_filter_gps_yofukashi.auf と vsd_plugins フォルダを AviUtl.exe と同じ場所にコピーしてください．

ほぼ必須：ffdshow のインストール

  1. こちらから ffdshow_rev3631_20101115_clsid.exe をダウンロードし，インストーラを実行します．最後まで「次へ」を押していけば OK です．
    ★重要★ ffdshow の rev3631 以前のバージョンでないと，あとで説明する圧縮 Codec が選択できません．上記のリンク以外から ffdshow を入手する場合は注意してください．

*.mp4 を読み込む場合：L-SMASH Works のインストール

  1. こちらの左上のリストから L-SMASH Works の最新版をダウンロードし，解凍してください．そして lwinput.aui を AviUtl.exe と同じ場所にコピーしてください．

MPEG-2 を読み込む場合：m2v.vfp のインストール

  1. こちらから m2v_vfp-???.lzh をダウンロードし，解凍してください．そして m2vconf.exe と m2v.vfp を AviUtl.exe と同じ場所にコピーしてください．
  2. コピーした m2vconf.exe を実行します．そのまま何もせず「決定」を押します．
  3. AviUtl.exe と同じ場所にコピーした m2v.vfp を m2v.aui にリネームします．
